# Python Version
Make sure you have Python 3.13 installed:
```bash
Python 3.13
```

# Required libraries
Install the required libraries:
```bash
python -m pip install pandas seaborn matplotlib scikit-learn openpyxl
```
# Files Structure
Ensure your project folder contains:
```bash
project-folder/
├── survey_analysis.py
├── RESULTS - Real Skills in a Virtual World - The Role of Video Game Genre Preferences in Young Adults' Social and Emotional Competence.xlsx
```
# Run
Run the Script:
```bash
python survey_analysis.py
```
