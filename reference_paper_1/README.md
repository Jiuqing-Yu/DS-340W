# Python Version
Make sure you have Python 3.10 installed:
```bash
Python 3.10
```

# Required libraries
Install the required libraries:
```bash
python -m pip install numpy pandas scipy tslearn umap-learn infomap networkx scikit-learn matplotlib
```
# Files Structure
Ensure your project folder contains:
```bash
project-folder/
├── time_series_pattern.py
├── Valve_Player_Data_with_genres.csv
```
# Run
Run the Script:
```bash
python time_series_pattern.py
```
